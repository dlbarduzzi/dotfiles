# vscode

## Installation

```sh
brew install --cask visual-studio-code
```

## Extensions

- [github-colortheme](https://marketplace.visualstudio.com/items?itemName=GitHub.github-vscode-theme)
- [golang](https://marketplace.visualstudio.com/items?itemName=golang.go)
- [python](https://marketplace.visualstudio.com/items?itemName=ms-python.python)
- [python-black-formatter](https://marketplace.visualstudio.com/items?itemName=ms-python.black-formatter)
- [python-isort](https://marketplace.visualstudio.com/items?itemName=ms-python.isort)
- [python-flake8](https://marketplace.visualstudio.com/items?itemName=ms-python.flake8)
- [prettier](https://marketplace.visualstudio.com/items?itemName=esbenp.prettier-vscode)
- [eslint](https://marketplace.visualstudio.com/items?itemName=dbaeumer.vscode-eslint)
- [tailwindcss](https://marketplace.visualstudio.com/items?itemName=bradlc.vscode-tailwindcss)
- [code-spell-checker](https://marketplace.visualstudio.com/items?itemName=streetsidesoftware.code-spell-checker)

## Settings

Copy file [settings.json](./settings.json) to `~/Library/Application\ Support/Code/User/settings.json` location.

## Dependencies

```sh
brew install golangci-lint
brew install gofumpt
```
